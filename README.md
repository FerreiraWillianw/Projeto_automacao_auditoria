# Auditoria Interna FUSVE

Sistema desenvolvido para automatizar o levantamento de processos e a matriz de riscos da Fundação Universidade de Vassouras (FUSVE). O sistema substitui o preenchimento manual de planilhas por um formulário web interativo conectado a um banco de dados relacional (PostgreSQL).

## 🚀 Funcionalidades
- **Diagnóstico de Processos:** Cadastro estruturado de informações da área e do processo.
- **Matriz de Riscos:** Mapeamento de riscos com cálculo automático de impacto e probabilidade.
- **Gestão de Dados:** Persistência em banco de dados SQL, permitindo consultas e relatórios centralizados.
- **Workflow de Aprovação:** Colunas dedicadas para validação da gerência e superintendência.

## 🛠 Tecnologias
- **Frontend:** [Streamlit](https://streamlit.io/)
- **Backend/Banco:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Integração:** SQLAlchemy
- **Linguagem:** Python 3.14
